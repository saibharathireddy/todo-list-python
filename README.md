# To-Do List Application using Python

This is a command-line based To-Do List application developed using Python. It helps users manage daily tasks efficiently.

## Features
- Add tasks
- View tasks
- Delete tasks
- Data is stored using file handling

## Technologies Used
- Python

## How to Run
1. Open the file in Python IDLE
2. Run the program
3. Use menu options to manage tasks
